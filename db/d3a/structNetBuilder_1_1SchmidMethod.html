Detailed Description

Class to compute the t-value of a projection of a digital net in base 2.

This class uses the algorithm described in [26], which consists, for each compositions of matrices, in enumerating all the combinations of the rows of the rows in the Gray code order, looking for a linear dependence between the columns.
