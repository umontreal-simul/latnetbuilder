LatNet Builder Manual

Overview

LatNet Builder is a software library and tool for constructing highly-uniform point sets for quasi Monte-Carlo and randomized quasi Monte-Carlo methods using state-of-the-art techniques. The software can construct rank-1 lattices and digital nets in base 2.

This software is based on a previous version called Lattice Builder. The reference for this previous version can be found below:

P. L'Ecuyer and D. Munger. LatticeBuilder: A General Software Tool for Constructing Rank-1 Lattice Rules. ACM Trans. Math. Softw. 42, 2, Article 15 (May 2016), 30 pages.

The current version of the software is called LatNet Builder, for Lattice & Net Builder, as it handles both rank-1 lattices and digital nets in base 2.
